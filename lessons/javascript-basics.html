JavaScript Variables

Variables are used to store information that can be referenced and manipulated in a program. They provide a way to label data with a descriptive name.

Declaring Variables

In JavaScript, you can declare variables using three different keywords:

var

var message = "Hello, World!";

The var keyword is the older way to declare variables. It has function scope.

let

let count = 10;

The let keyword was introduced in ES6. It has block scope, which makes it preferable to var in most cases.

const

const PI = 3.14159;

The const keyword is used for variables that won't be reassigned. It also has block scope.

Variable Naming Rules

  • Names can contain letters, digits, underscores, and dollar signs
  • Names must begin with a letter, $, or _
  • Names are case sensitive (y and Y are different variables)
  • Reserved words (like JavaScript keywords) cannot be used as names

Examples

// Number variable
let age = 25;

// String variable
let name = "John Doe";

// Boolean variable
let isStudent = true;

// Array variable
let colors = ["red", "green", "blue"];

// Object variable
let person = {
  firstName: "Jane",
  lastName: "Doe",
  age: 30
};
